Adds a new book to the inventory. 

This function interacts with the user to populate all fields of a new \doxylink{classbookInfo}{book\+Info} record (title, author, ISBN, etc.). It performs validation on numeric fields, checks for duplicate ISBNs, and only appends the record to the inventory if the user chooses to save it.

\hline
{\em capacity} & Maximum allowed number of books in the system (default = 20).\\
\hline

\begin{DoxyPostcond}{Postcondition}

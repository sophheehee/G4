\doxysubsection{Detailed Description}
Represents a single book entry in the Serendipity inventory. 

A \doxylink{classbookInfo}{book\+Info} object stores book metadata including identifying information (title, author, ISBN), publication information (publisher, date added), and inventory data (quantity on hand, wholesale and retail prices).

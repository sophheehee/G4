void editBook ( std::vector< bookInfo > & inventory)

Allows full editing of a selected book record.

The user selects a book (via lookup), views all fields, and can modify each field (title, ISBN, author, etc.) one at a time. Changes are staged in a temporary record and applied only when the user selects "Save".

Parameters
inventoryReference to the inventory vector.
Postcondition
If saved, the selected entry in inventory is updated.

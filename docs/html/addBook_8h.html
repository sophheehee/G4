Detailed Description

Declares the addBook function for inserting new book records into the Serendipity inventory system.

The addBook function provides a menu-driven interface for entering book information, validating data, preventing duplicate ISBNs, and appending a completed book record into the inventory vector.

void addBook ( std::vector< bookInfo > & inventory,
std::size_t capacity = 20 )

Adds a new book to the inventory.

This function interacts with the user to populate all fields of a new bookInfo record (title, author, ISBN, etc.). It performs validation on numeric fields, checks for duplicate ISBNs, and only appends the record to the inventory if the user chooses to save it.

Parameters
inventoryReference to the vector storing all book records.
capacityMaximum allowed number of books in the system (default = 20).
Precondition
inventory.size() <= capacity
Postcondition
If saved, a fully populated bookInfo record is appended to inventory.

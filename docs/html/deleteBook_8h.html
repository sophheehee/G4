void deleteBook ( std::vector< bookInfo > & database)

Deletes a book from the inventory.

This function allows the user to search for a book, select it, display its details, and confirm deletion. If confirmed, the record is erased from the vector and remaining entries shift to fill the gap.

Parameters
databaseReference to the inventory vector.
Postcondition
If confirmed, inventory shrinks by one entry.
